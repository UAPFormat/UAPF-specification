# uapf-cli (Proof of Concept)

Minimal tooling to make the spec implementable.

Commands:
- `python uapf.py validate <path>`  (validate example packages and enterprise index)
- `python uapf.py pack <package-dir> <out.uapf>`
- `python uapf.py unpack <in.uapf> <out-dir>`

This is a reference implementation and not a runtime engine.
